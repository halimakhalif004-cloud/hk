introduction to web design

web designis a craft of shapping the look and the feel of the app or the website it is overall layout and the colour the user can seeit.

Web design focuses mainly on how a website looks and how users interact with it. Designers choose the layout, colors, fonts, images, and overall style to create a pleasant and user-friendly experience. They also consider user interface (UI) and user experience (UX) principles to make sure visitors can easily navigat the site, find information quickly, and enjoy using the website on different devices such as phones, tablets, and computers

what is web development

Web development deals with the coding and technical structure that brings the design to life. Front-end development uses languages like HTML, CSS, and JavaScript to create the parts of the website that users see and interact with. Back-end development works behind the scenes using servers, databases, and programming languages such as Python, PHP, Java, or Node.js to manage data, handle user logins, and support website functionality.

the web development process includes web design, web content development,client-side/server-side scripting and network security configuration,among other task

the three core of technology

HTML _ HyperText Markup Language

is the basic structure of a web page. It is used to create headings, paragraphs, images, links, and other content. Without HTML, a webpage would not have any organized structure.

CSS _ Cascading Style Sheet

is used to style and design the webpage. It controls colors, fonts, layouts, spacing, and how the page looks on different screen sizes. CSS makes websites attractive and visually appealing

JavaScript

JavaScriptis a programminglanguage that conforms to the ECMAScript specification. JavaScript is high_level, often just-in-time compiled,and multi-paradigm.
